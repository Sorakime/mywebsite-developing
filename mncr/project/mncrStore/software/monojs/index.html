monochrome.から、新しいJavaScriptのフレームワークが登場します。

その名も、monoJS。

これまでよりもずっとスマートなフレームワーク。より短い記述で、より軽くウェブ開発ができます。

また、Mono.jsというサーバーサイド向けJSフレームワークがありますが、それとこれはまったくの別物です。


ベータ1なので、シンプルな機能しかないですが、もっとどんどん増やしていく予定です。
